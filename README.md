# Coin Screener Dashboard

一个静态的选币与分析工具，界面参考交易驾驶舱风格：

- Binance 公开行情为默认数据源，OKX 可作为备选切换
- USDT 现货与 USDT 永续合约标的筛选
- 24h 涨跌幅、成交额、持仓量变化、主动买卖差、资金费率、现货突发买盘、热度分排序
- 拉盘雷达分：短周期涨幅、量能突增、主动买卖差、现货大单、持仓量、资金费率、突破结构和流通市值综合评分
- 代币总量、流通总量、流通市值、FDV 展示
- 单币种 5m、15m、30m、1H、4H、1D K 线、突破位、保护位、止盈区、RSI、ATR 和量能确认
- K 线画布支持滚轮缩放和拖动平移
- 自动刷新和接口失败样例数据

交易数据来自 Binance / OKX 公开行情接口，供应量和市值数据来自 CoinGecko 公开市场数据。页面只读取公开行情，不连接账户，不下单。
